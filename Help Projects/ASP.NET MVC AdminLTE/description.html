Introduction

A Visual Studio 2015 project which shows how use create an Asp.Net MVC Application with a custom layout. The AdminLTE layout (MIT License) from https://code.msdn.microsoft.com/ASPNET-MVC-Application-b4b0dc3f/https://code.msdn.microsoft.com/ASPNET-MVC-Application-b4b0dc3f/https://almsaeedstudio.com/preview has been selected for the customization.

The project illustrates the following topics:

  • Adding an existing layout to an Asp.Net MVC application.
  • Adding 3rd party JavaScript plugins

Building the Sample

  1. Extract the package WebApplication.zip
  2. Double-click on the WebApplication.sln
  3. Click F5 to start debugging

Description

Web Application requires user authorization, click "Register as a new user" link and create an account.
